# ZOG DBF

## Getting Started

### Go

```bash
go mod tidy

# Create DBF
go run create/create.go

# Read DBF
go run read/read.go
```

### Python

```bash
uv sync


# Create .dbf
uv run python create_dbf.py

# Read .dbf
uv run python read_dbf.py
```

## Conclusion

Overall, it depends on the DBF file that we use.
